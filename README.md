# House Price Prediction

## XYLOFY AI: Internship - Week 1

### Project Overview

This project aims to predict house prices using machine learning techniques based on various housing features such as area, bedrooms, bathrooms, parking availability, furnishing status, and other property characteristics.

### Dataset

The dataset contains information about residential properties and their corresponding prices.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

### Project Workflow

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Encoding using One-Hot Encoding
4. Model Building

   * Linear Regression
   * Random Forest Regressor
5. Model Evaluation
6. Data Visualization
7. Insights and Recommendations

### Model Performance

#### Linear Regression

* MAE: 970,043
* RMSE: 1,324,506
* R² Score: 0.652

#### Random Forest Regressor

* MAE: 1,021,546
* RMSE: 1,400,565
* R² Score: 0.611

### Key Findings

* Area was the most influential feature affecting house prices.
* Bathrooms, air conditioning, parking availability, and number of stories also contributed significantly.
* Linear Regression outperformed Random Forest on this dataset.

### Conclusion

The project demonstrates how machine learning can be applied to predict house prices and identify the factors that influence property valuation.
